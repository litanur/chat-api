## General Purpose

Backend Technical Test from PT. Pilar Teknologi

## Description

It is a chat on Nest.js(Passport(JWT+Local)), Socket.io and MongoDB(Mongoose). You can:
1. Login
2. Register
3. Create Room Chat
4. Send Message
5. Get All Message by chat_id

See OpenAPI (Swagger) http://localhost:3000/api/#/

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev
```
